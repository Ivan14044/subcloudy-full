## 🔧 АЛЬТЕРНАТИВНЫЙ СПОСОБ (вручную):

Если батник не работает:

### **Шаг 1: Скопируйте флаги**

Откройте CMD:
```cmd
cd D:\project\Subcloudy\subcloudy-desktop

copy ..\ai-bot-front-main\public\img\lang\ru.png dist\renderer\img\ru.png

copy ..\ai-bot-front-main\public\img\lang\en.png dist\renderer\img\en.png

copy ..\ai-bot-front-main\public\img\lang\uk.png dist\renderer\img\uk.png
```

### **Шаг 2: Проверьте**
```cmd
dir dist\renderer\img\*.png
```

Должно показать:
```
google.png
telegram.png
ru.png
en.png
uk.png
```

### **Шаг 3: Запустите**
```cmd
npm run dev
```

## 🐛 ЕСЛИ ВСЁ ЕЩЁ НЕ РАБОТАЕТ:

### **Полная очистка и пересборка:**

```cmd
cd D:\project\Subcloudy\subcloudy-desktop

REM 1. Остановить Electron
taskkill /F /IM electron.exe

REM 2. Удалить dist
rmdir /S /Q dist
rmdir /S /Q dist-electron

REM 3. Пересобрать
npm run build

REM 4. Скопировать флаги
copy ..\ai-bot-front-main\public\img\lang\*.png dist\renderer\img\

REM 5. Запустить
npm run dev
```
