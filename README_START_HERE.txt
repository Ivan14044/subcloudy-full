🚀 ДЛЯ ЗАПУСКА ВСЕГО ПРОЕКТА:

   1. Откройте проводник Windows
   2. Перейдите в папку: D:\project\Subcloudy
   3. Дважды кликните на файл: LAUNCH_EVERYTHING.bat
   4. Подождите 10-15 секунд
   5. Откроется браузер с фронтендом!

───────────────────────────────────────────────────────────────

📍 ЧТО ОТКРОЕТСЯ:

   ✅ 2 окна CMD:
      - Зелёное = Backend (Laravel)  
      - Жёлтое = Frontend (Vite)
   
   ✅ Браузер с адресом: http://localhost:5173

───────────────────────────────────────────────────────────────

🔗 ССЫЛКИ ДЛЯ ТЕСТИРОВАНИЯ:

   📌 Frontend (публичный сайт):
      http://localhost:5173

   📌 Backend API:
      http://127.0.0.1:8000/api

   📌 Admin панель:
      http://127.0.0.1:8000/login

   📌 Desktop приложение:
      subcloudy-desktop\START_APP.bat

───────────────────────────────────────────────────────────────

⚠️ ВАЖНО:

   ❗ НЕ ЗАКРЫВАЙТЕ цветные окна CMD!
   ❗ Это backend и frontend серверы
   ❗ Закроете - всё остановится

───────────────────────────────────────────────────────────────

🛠️ ЕСЛИ ЧТО-ТО НЕ РАБОТАЕТ:

   1. Закройте ВСЕ окна CMD
   2. Запустите LAUNCH_EVERYTHING.bat снова
   3. Подождите 10-15 секунд
   4. Обновите страницу в браузере (F5)
